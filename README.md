# czsh

czsh is a minimal shell written in C.

## Built-in Commands

- `cd`
- `help`
- `exit`

## Getting Started

### Dependencies

- `gcc` or any C compiler that compiles for C99.

### Usage

- Clone the repository and run `make` to compile.
- Run `make run` or `./czsh` to start the shell.
